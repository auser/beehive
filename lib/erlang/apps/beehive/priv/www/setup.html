Welcome to Beehive. Clearly you haven't customized the Beehive setup. The instructions to do so are as follows:

  • Create a configuration file (in $BEEHIVE_HOME/etc/beehive.conf or /etc/beehive.conf)
  • Add the line:
    base_app: name
  • Add the base app
    curl -i -XPOST -d"{\"name\":\"name\", \"url\":\"url\",
    \"token\":$TOKEN}" $HOST/apps
  • That's it!
